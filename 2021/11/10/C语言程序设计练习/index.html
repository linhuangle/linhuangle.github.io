C语言程序设计练习

1.输入一个圆的半径r（0<r≤1000)，圆周率PI为3.14159，计算并输出该圆的面积(保留一位小数)。

#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
int r;
float s, p = 3.14159;
scanf(“%d”, &r);
if (r <= 0 || r > 1000)
printf(“输入的半径不在限定范围内\n”);
else {
s = r * r * p;
printf(“%.1f”, s);
}
return 0;
}

2.输入两个正整数，输出其求差算式。

#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
int a, b, c;
scanf(“%d %d”, &a, &b);
c = a - b;
printf(“%d-%d=%d”, a, b, c);
return 0;
}

3.输入一个十六进制整数，输出其八进制、十进制无符号数。

#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
int a;
scanf(“%x”, &a);
printf(“%o %u”, a, a);
return 0;
}

4.输入格式为“整数字符串整数，整数”的数据，其中字符串长度为固定值5，将三个整数求和输出。

#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
int a, b, c;
scanf(“%d%*5c%d,%d”, &a, &b, &c);
printf(“%d”, a + b + c);
return 0;
}
