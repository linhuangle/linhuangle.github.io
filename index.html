My First Blog

<–**C语言基础学习笔记**–>

一个C程序一般由若干头文件及函数组成。

以输出“Hello World”为例

#include<stdio.h>
int main()
{
printf(“Hello World\n”);
return 0;
}

其中#include<stdio.h>便是一条预处理命令，其作用为通知C语言编译系统在对C程序进行正式编译之前需做一些预处理工作，包含了头文件。

而int main()
{
printf(“Hello World\n”);
return 0;
}即函数，其为实现代码逻辑的一个小单元。

int main()
{
printf(“Hello World\n”);
return 0;
}

知识点

“int main()”是C语言中唯一的主函数

“printf()”是格式输出函数

“return”是函数的返回值，根据函数类型的不同，返回值也是不同的

“\n”是转义字符中的换行符

良好习惯

一个说明或一个语句占一行

括号要成对写，如需删除亦需成对删除

当一句可执行语句结束时，末尾应由分号结尾

代码中所有符号均应为英文半角符号
